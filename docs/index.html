DEVAN is an MCP-orchestrated agent framework for document intelligence and data pipelines — designed to be modular, extensible, and production-ready.

Why DEVAN?

⚡

MCP-Orchestrated

Built natively on the Model Context Protocol — agents compose via standard tool interfaces, no glue code required.

📄

Document Intelligence

First-class support for PDFs, Word, Excel, PowerPoint, and OpenSearch — extract, transform, and reason over documents.

🔧

Composable Pipelines

Chain agents into data pipelines with minimal configuration. Each server is independently deployable and testable.

🔒

Production-Ready

Apache 2.0 licensed, with a security policy, CodeQL scanning, and Dependabot keeping dependencies fresh.
